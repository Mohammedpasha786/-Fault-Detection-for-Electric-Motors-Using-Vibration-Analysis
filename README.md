# Fault Detection for Electric Motors Using Vibration Analysis

A MATLAB/Simulink model-based design system that detects electric motor faults from vibration data and deploys a real-time classifier onto the **STM32 NUCLEO-H743ZI2** board via Simulink Coder.

Industry partner: **STMicroelectronics**

## Quick Start

```matlab
% 1. Add all source folders to path
addpath(genpath('src'));

% 2. Generate synthetic vibration data for all fault classes
generateMotorData();          % writes CSVs to data/synthetic/

% 3. Preprocess + extract features + train + evaluate
run scripts/run_training_pipeline.m

% 4. One-shot demo with plots
run scripts/run_demo.m
```

## Fault Classes Simulated

| Class | ID | Simulation approach |
|---|---|---|
| Normal | 0 | Baseline BLDC motor vibration |
| Bearing fault | 1 | BPFO/BPFI tonal noise at bearing frequencies |
| Rotor imbalance | 2 | 1× running-speed sinusoid added to all axes |
| Shaft misalignment | 3 | 2× running-speed harmonic + cross-axis coupling |
| Electrical fault | 4 | Supply-frequency harmonic + amplitude modulation |

## Requirements

| Requirement | Notes |
|---|---|
| MATLAB R2023b+ | Core language |
| Statistics and Machine Learning Toolbox | `fitcsvm`, `fitcensemble`, `confusionchart` |
| Signal Processing Toolbox | `bandpass`, `envelope`, `fft` wrappers |
| Predictive Maintenance Toolbox | Feature extraction helpers (optional — custom fallbacks provided) |
| Simulink + Simulink Coder | STM32 code generation |
| Simulink Coder Support Package for STMicroelectronics Nucleo Boards | STM32 deployment |
| STM32 NUCLEO-H743ZI2 board + LIS3DH accelerometer | Hardware (optional) |

`generateMotorData`, `preprocessVibration`, all feature extractors, and all tests run on **base MATLAB + Signal Processing Toolbox** only.

## Deployment to STM32

See `src/deployment/stm32DeploymentNotes.m` and `docs/ROADMAP.md` Milestone 5 for the full checklist. The short version:

1. Train and export the classifier: `exportModelForCodegen(mdl, 'FaultClassifier')`
2. Open the Simulink model, point the MATLAB Function block at the exported classifier.
3. Set the target to `STM32 NUCLEO-H743ZI2` in Simulink Coder.
4. Wire the LIS3DH accelerometer block (I2C) and LED/buzzer output blocks.
5. Build and deploy.
